# Vinamra Duvey — Personal Portfolio

A modern, AI-themed personal portfolio website for [vinamra.me](https://vinamra.me) built with React + Vite + Tailwind CSS.

## Features

- **Portfolio page** — Hero, About, Professional Journey (timeline), Education, Skills & Expertise, Projects preview, Vision, Contact
- **Projects page** — Detailed case studies for all Railway AI projects with expandable write-ups:
  - 📊 **Depot IQ** — Real-Time Depot Occupancy Intelligence Platform
  - 🚆 **Train IQ** — AI-Powered Train Health & Performance Monitor
  - ⚙️ **StartM** — Smart Maintenance Workflow Automation
  - 📈 **Predictive Maintenance AI** — Data-Driven Asset Health Forecasting
  - 🔧 **FRACAS Pro** — AI-Enhanced Failure Reporting & Corrective Action System
  - 🛡️ **RAMS Analytics** — AI-Enhanced RAMS Platform (Research Concept)
  - 🤖 **Oculanet AI Suite** — Commercial Chatbot & Automation Solutions

## Tech Stack

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [React Router v6](https://reactrouter.com/)
- [Lucide React](https://lucide.dev/)

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built files will be in the `dist/` folder — deploy to any static host (Netlify, Vercel, Cloudflare Pages, GitHub Pages).

## Deployment

### Netlify / Cloudflare Pages
Point the build directory to `dist/` — SPA redirects are handled by `public/_redirects`.

### GitHub Pages
Update `vite.config.js` to set `base: '/vinamra/'` (or your repo name), then deploy the `dist/` folder.
